## Email Best Practices

1. **Subject lines:** Keep under 50 chars
2. **Greeting:** Use first name only (Hi [Name])
3. **Body:** Max 150 words before CTA
4. **CTA:** One link per email (Calendly)
5. **Signature:** Name + company + one optional link
6. **Cadence:** Send 50 emails/day max (avoid spam filters)
7. **Timing:** Send 9-11am on Tuesday-Thursday
8. **Pause:** Wait 3 days between follow-ups
